Участки с результатами нейросетевого моделирования

Переключатель типа карты

Кнопка в виде стопки листов (верхний правый угол) открывает список из нескольких режимов отображения — Схема, Спутник, Гибрид и Народная карта.

Элементы масштабирования

Кнопки плюс-минус позволяют менять разрешение карты с определенным шагом. Для маштабирования можно использовать колесо мыши

Линейка

Что бы измерить расстояние - щелкните линейку справа внизу и наносите точки на карте.

Выбор участка по нумерованной метке

Информация об объектах появляется в виде всплывающей гиперссылки и не закрывает полезную область карты около метки

Карта на весь экран

Кнопка две стрелки (верхний правый угол) раскрывает карту на весь экран браузера. Повторное нажатие востанавливает исходный размер

Поиск по карте

Поиск фрагмента карты по городскому адресу, названию населенного пункта или по данным широты и долготы
